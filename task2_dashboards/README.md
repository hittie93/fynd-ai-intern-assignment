# Task 2: Feedback AI Ecosystem

This folder contains the Streamlit application for the AI Feedback System.

## Architecture
- `app.py`: Main application entry point.
- `data/feedback.csv`: CSV storage for feedback data.
- `prompts/`: Directory for prompt templates (optional/future use).

## Running the App
From the project root:
```bash
streamlit run task2_dashboards/app.py
```
